Instructions

  1. Press "SELECT ROM" to load your GameBoy ROM file
  2. If you have a savefile, press "SELECT SAVE" to load savefile
  3. Press "LOAD" to play the game
  4. Press "DOWNLOAD SAVE" to download a savefile of the current game state

Keyboard Mappings

A & S for buttons A & B

← ↑ → ↓ for directions

Spacebar for Select

Enter↩ for Start
